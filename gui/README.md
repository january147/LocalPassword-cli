# 密码管理器 GUI

基于 Tkinter 的图形界面，为密码管理器提供友好的用户界面。

## 🚀 快速开始

### Linux/Mac

```bash
# 给启动脚本添加执行权限
chmod +x run_gui.sh

# 运行 GUI
./run_gui.sh
```

### Windows

双击运行 `run_gui.bat` 文件。

### 直接运行 Python 脚本

```bash
python3 gui/password_manager_gui.py
```

## 📋 功能特性

### 核心功能

- ✅ **添加密码** - 创建新的密码条目
- ✅ **编辑密码** - 修改现有的密码条目
- ✅ **删除密码** - 删除不需要的密码
- ✅ **搜索密码** - 快速搜索密码列表
- ✅ **复制密码** - 一键复制密码到剪贴板
- ✅ **生成密码** - 生成强密码
- ✅ **刷新列表** - 刷新密码列表
- ✅ **导出数据** - 导出密码为 JSON 格式
- ✅ **导入数据** - 从 JSON 文件导入密码

### 密码字段

每个密码条目包含以下字段：

- **标题** - 密码条目的名称（必填）
- **用户名** - 登录用户名
- **密码** - 登录密码（加密存储）
- **网站** - 相关网站 URL
- **备注** - 附加说明

### 密码生成器

功能强大的密码生成器：

- 可自定义密码长度（8-64 位）
- 可选择字符类型：
  - 大写字母 (A-Z)
  - 小写字母 (a-z)
  - 数字 (0-9)
  - 特殊字符 (!@#$%^&*)
- 一键生成并复制到剪贴板

## 🔧 技术细节

### 依赖

- Python 3.6+
- Tkinter (Python 标准库，无需额外安装）

### 架构

```
GUI (Tkinter)
    ↓
CLI 调用 (subprocess)
    ↓
密码管理器 (Rust)
    ↓
加密数据库 (SQLite)
```

### 安全性

- 密码使用 AES-256-GCM 加密
- 密钥派生使用 Argon2id
- 主密码不会在内存中明文存储
- 数据库文件加密存储

## 🐛 故障排除

### 问题: 找不到密码管理器可执行文件

**解决方案**:
```bash
# 确保已编译密码管理器
cd /workspace/projects/workspace/password-manager
cargo build --release
```

### 问题: GUI 界面显示异常

**解决方案**:
- 确保安装了 Python 3.6+
- 确保 Tkinter 已安装
  ```bash
  python3 -c "import tkinter; print('OK')"
  ```

### 问题: 密码列表为空

**解决方案**:
1. 确保数据库已初始化
2. 输入正确的主密码解锁数据库

## 📝 使用示例

### 1. 创建新数据库

1. 点击 "文件" → "新建数据库"
2. 输入主密码
3. 点击 "确定"

### 2. 添加密码

1. 点击 "添加" 按钮
2. 填写密码信息
3. 点击 "保存"

### 3. 搜索密码

1. 在搜索框中输入关键词
2. 列表自动过滤显示匹配结果

### 4. 生成密码

1. 点击 "生成密码" 按钮
2. 设置密码长度和选项
3. 点击 "生成"
4. 密码自动复制到剪贴板

### 5. 导出数据

1. 点击 "文件" → "导出"
2. 选择保存位置
3. 数据导出为 JSON 文件
