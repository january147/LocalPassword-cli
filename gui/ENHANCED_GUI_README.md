## 🚀 使用方法

### Linux/Mac

```bash
# 给启动脚本添加执行权限
chmod +x run_gui_v2.sh

# 运行增强版 GUI
./run_gui_v2.sh
```

### Windows

双击运行 `run_gui_v2.bat` 文件。

### 直接运行 Python 脚本

```bash
python3 gui/password_manager_gui_v2.py
```

## 🔧 技术实现

### 模块化设计

```
gui/
├── password_manager_gui_v2.py  # 主程序（33 KB）
├── password_manager_gui.py      # 原版（20 KB）
├── welcome_page.py               # 欢迎页面
├── settings_dialog.py            # 设置对话框
├── toast_manager.py             # Toast 通知
├── theme_manager.py              # 主题管理器
└── password_strength.py          # 密码强度分析器
```

### 核心模块

1. **PasswordManagerGUI** - 主 GUI 类
2. **ThemeManager** - 主题管理（亮色/暗色）
3. **ToastManager** - Toast 通知系统
4. **PasswordStrengthAnalyzer** - 密码强度分析
5. **WelcomePage** - 欢迎页面
6. **SettingsDialog** - 设置对话框
7. **PasswordDialog** - 密码对话框
8. **PasswordGeneratorDialog** - 密码生成器
